====test====
Prism is able to respond with an empty response when authorization fails.
====spec====
swagger: "2.0"
paths:
  /users:
    get:
      description: "abc"
      responses:
        401:
          description: "abc"
securityDefinitions:
  Basic:
    type: basic
security:
  - Basic: []
====server====
mock -p 4010
====command====
curl -sIXGET http://localhost:4010/users
====expect====
HTTP/1.1 401 Unauthorized
