====test====
When I send a request to an operation
And in the path I specify ?__code=404
And the operation has no 404 status code response defined
And the operation has 400 status code defined, but no schema or example defined in the response
Then I get back a 404 response with an error code
====spec====
{
  "info": {
    "_postman_id": "37c4cd69-28e5-48aa-8386-a4ec010216e4"
  },
  "item": [
    {
      "id": "3b4f5a70-2a8f-46ba-a04b-a6f80621ad3f",
      "request": {
        "url": {
          "path": ["todos"]
        },
        "method": "GET"
      },
      "response": [
        {
          "description": "test",
          "code": 400
        }
      ]
    }
  ]
}
====server====
mock -p 4010 ${document}
====command====
curl -i -X GET http://localhost:4010/todos?__code=404
====expect====
